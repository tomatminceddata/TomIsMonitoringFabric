## Power BI Tenant Lens
Power BI Tenant Lens helps me create an inventory of the Power BI items currently available in my tenant.
A PowerShell script is used to extract metadata from my tenant. This script writes CSV files to the folder where the PowerShell script is located. Power BI template file allows me to import the CSV files easily and I can start analyzing the Power BI items.
Please read the documents:
+ PowerBITenantLens_Guide.md, describes the PowerShell script and
+ Power BI Tenant Lens - the Power BI solution.md, describes the Power BI solution
